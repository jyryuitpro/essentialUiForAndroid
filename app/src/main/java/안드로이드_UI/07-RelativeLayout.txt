RelativeLayout
- 부모 뷰가 될 수 있는 컨테이너 뷰이다
- 자식뷰의 배치를 설정 할 수 있다
    - 부모를 기준으로 어디로 이동
    - 기준점을 정해놓고 어디로 이동
    - 기준점을 중심으로 자식뷰를 배치
        - 기준점 : 부모, 자식뷰
    - 기준점을 표시하는 방법
        - id속성을 사용
    - 기준점을 여러개 잡을 수 있다
    A B C D
    B : A를 기준으로 A의 오른쪽
    C : D를 지준으로 D의 왼쪽

    A는 B의 왼쪽 B는 C의 왼쪽 C는 "D"의 왼쪽 (문제가 있는 뷰 구조)
